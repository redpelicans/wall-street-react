# Wall Street React

Sandbox application in [React](http://facebook.github.io/react/) about stock exchange.

## Setup

    npm install

## Usage

    npm start
    npm run lint

## Misc

### File structure

* app/
  * index.js _(entry point)_
  * app.js
  * components/
* public/
  * index.html
  * styles/
  * images/
  * fonts/
* build/
* ... _(config files like `webpack.config.js` and dev files like `server.js`)_
